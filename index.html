Google Analytics Event Tracking Plugin for jQuery

Add Google Analytics event tracking to your elements with this handy jQuery plugin

View project onGitHub

Want to quickly track user interactions on your website?

This plugin, a work-in-progress by @tagr, provides a method to add event tracking to any element(s) you wish with jQuery.

GAEvent is a small plugin that enables event tracking in Google Analytics (by ga.js or the newer Universal Analytics)

The syntax to start tracking is very simple:

$('.yourElements').gaEvent();

If you want to customize the data sent to Google, just add the options like so:

$('.yourElements').gaEvent({
  action: 'tap',
  category: 'Google Analytics Event Tracking Category',
  label: 'Custom Label',
  value: 1.25
});
